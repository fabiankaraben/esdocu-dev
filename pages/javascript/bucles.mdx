# Bucles

Hemos visto cómo usar métodos de matriz como forEach y map para iterar sobre los elementos de una matriz. En programación, a esto lo llamamos bucle. Hacer un bucle significa repetir un proceso hasta que se cumpla una condición. Podría ser hasta que se complete una cantidad específica de veces, se complete una acción específica o se alcance un valor. Es un concepto fundamental en programación y se utiliza de varias maneras. JavaScript tiene diferentes tipos de bucles, además de los métodos de matriz que hemos visto, que permiten un mayor control sobre el proceso de bucle. Estos son el bucle for, el bucle while y el bucle do... while. Echemos un vistazo a cada uno de estos.
