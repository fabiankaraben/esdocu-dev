# ¿Cómo escribir y ejecutar código en JavaScript?

La mejor manera de aprender JavaScript es escribiendo JavaScript. Exploremos las diferentes formas en que puedes escribir y ejecutar código JavaScript para que te resulte fácil practicar todo lo que aprendas.

## Usando un navegador web

Una de las formas más sencillas y accesibles de escribir código JavaScript es mediante un navegador web. La mayoría de los navegadores web modernos, como Google Chrome, Mozilla Firefox y Safari, tienen un motor JavaScript integrado que te permite escribir y ejecutar código JavaScript directamente en el navegador.

Para escribir código JavaScript en un navegador web, puedes utilizar las herramientas de desarrollo del navegador. Para abrir las herramientas de desarrollo en Google Chrome, haz clic derecho en cualquier página web y selecciona "Inspeccionar" en el menú contextual. Alternativamente, puedes presionar `Ctrl + Shift + I` en una computadora con Windows o Linux, o `Comando + Opción + I` en una Mac.

Una vez que la ventana de herramientas de desarrollador esté abierta, navega hasta la pestaña "Consola", donde podrás escribir tu código JavaScript. Por ejemplo, puedes intentar escribir el siguiente código en la consola, pero con tu nombre en lugar del mío:

```javascript filename="JavaScript"
console.log('Mi nombre es Esdocu.')
console.log('¡JavaScript es increíble!')
```

Para ejecutar su código JavaScript, presione la tecla Enter en su teclado. Esto imprimirá el mensaje "Mi nombre es Ebenezer Don" y "¡JavaScript es increíble!" a la consola. ¡Buen trabajo! Explicaré cada parte del código que acabas de escribir más adelante.

## Usando un editor de código en línea

Otra forma de escribir y ejecutar código JavaScript es mediante el uso de un editor de código en línea o un entorno de desarrollo integrado (IDE) en línea. Hay muchos editores de código e IDE en línea disponibles que te permiten escribir y ejecutar código JavaScript desde tu navegador web. Algunas opciones populares incluyen CodePen, JSFiddle y JS Bin.

Necesitas una conexión a Internet para utilizar un editor de código en línea, pero también tienes varias ventajas. Primero, te permite escribir y ejecutar código desde cualquier lugar. En segundo lugar, puedes compartir fácilmente tu código con otras personas a través de enlaces. En tercer lugar, muchos editores de código en línea e IDE tienen funciones integradas que pueden facilitar el proceso de escritura de código y verificación de errores.

## Usando un editor de código local

También hay varios editores de código sin conexión e IDE que puedes utilizar para escribir y ejecutar código JavaScript. Algunas opciones populares incluyen Eclipse, NetBeans, Visual Studio y Visual Studio Code (con la ayuda de extensiones). Estos IDE deben instalarse en tu computadora local y, por lo general, tienen más características y funcionalidades que los editores de código en línea. A menudo se utilizan para proyectos más complejos e incluso si no comienzas con ellos, eventualmente necesitarás uno a medida que avances.

La forma más fácil de comenzar es la consola de tu navegador, por lo que recomiendo escribir y ejecutar tu primer código JavaScript en ella.
