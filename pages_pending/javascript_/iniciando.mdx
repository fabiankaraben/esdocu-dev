# Iniciando con JavaScript

JavaScript todavía se utiliza principalmente para crear páginas web dinámicas dentro de un navegador. Antes de analizar más de cerca otras áreas de aplicación en capítulos posteriores, este capítulo te mostrará las formas en que puedes incrustar JavaScript en una página web y generar resultados simples. Por lo tanto, este capítulo es la base para los siguientes.

Antes de entrar en más detalles sobre el lenguaje JavaScript en sí, primero debes saber cómo se relaciona JavaScript con HTML y CSS dentro de una página web, cómo incrustar JavaScript en una página web y cómo generar resultados.
