# Introducción a JavaScript y desarrollo web

Los tres lenguajes más importantes para crear interfaces web son, sin duda, HTML, CSS y JavaScript. Cada uno de estos lenguajes tiene su propio propósito.

## La relación entre HTML, CSS y JavaScript

En HTML, se utilizan elementos HTML para especificar la estructura de una página web y el significado (semántica) de los componentes individuales de una página web. Por ejemplo, describen qué área de la página web es el contenido principal y qué área se utiliza para la navegación, y definen componentes como formularios, listas, botones, campos de entrada o tablas.

CSS, por otro lado, utiliza reglas CSS especiales para determinar cómo se deben mostrar los componentes individuales que has definido previamente en HTML; esto se utiliza para definir el diseño y la disposición de una página web. Por ejemplo, puedes definir el color y el tamaño del texto, los bordes, los colores de fondo, los degradados de color, etc.
